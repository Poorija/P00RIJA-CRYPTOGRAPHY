## 🌐 English

### 🚀 Overview
**P00RIJA Cryptography** is a cutting-edge, client-side security suite designed for modern privacy needs. It provides a comprehensive set of tools for encryption, secure communication, and system monitoring, all delivered through a unified native and web experience.

<div class="card">
  <h3>💎 Key Features</h3>
  <ul>
    <li><span class="feature-badge">Native</span> <b>Tauri 2.11.1</b> Desktop Shell (Windows, macOS, Linux).</li>
    <li><span class="feature-badge">Web</span> Progressive Web App (PWA) support.</li>
    <li><span class="feature-badge">Security</span> Client-side Web Crypto (AES-GCM, RSA, ECC).</li>
    <li><span class="feature-badge">Chat</span> PeerJS/WebRTC Encrypted Chat with Video/Voice support.</li>
    <li><span class="feature-badge">Build</span> New <b>Graphical Build Wizard</b> with x86_64 and ARM64 support.</li>
  </ul>
</div>

### 🛠 Installation & Setup
1. **Install Dependencies:**
   ```bash
   npm install
   ```
2. **Setup Environment:**
   ```bash
   npm run setup:macos  # For macOS
   npm run setup:linux  # For Linux
   npm run setup:windows # For Windows
   ```
3. **Launch the Build Wizard:**
   ```bash
   npm run native:build:wizard
   ```
